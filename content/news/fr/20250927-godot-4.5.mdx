# Godot 4.5 !

Oyez! Oyez!

Les builds de godot pour Windows, MacOS et Linux sont disponibles avec les bonnes options de compilation. Les builds sont désormais compilés automatiquement par la CI de github (les fichiers d'exports ne sont pas encore prêt, mais pas d'urgence)

Vous les trouverez ici : https://github.com/DyingStar-game/godotandaddons/releases/tag/4.5-stable

Le projet DyingStar (godot) n'est pas encore en 4.5 (branche main) mais le sera d'ici à quelques heures (merge de la branche `feature/HorizonServer` dans la branche `main` en cours de revue de code ( = vérification du code par un autre développeur))
